# Grip - Log and Share your Climbing Moments <br />
https://get-a-grip.herokuapp.com/ <br />
Instagram inspired application for climbers and boulderers to write brief logs about their achievements or experiences. Features Google Maps API to track locations. Saved images through Amazon Web Service S3. <br />
